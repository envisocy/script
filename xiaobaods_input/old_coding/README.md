### 程序运行

直接调用xiaobaods_input包即可。
- 生意参谋商品信息系统：读取信息路径需设定目录于Business_adviser_parser/config.py文件，文件名为html.txt。
- 生意参谋关键词系统：读取信息路径需设定目录下config.py文件，文件名为默认下载csv文件；
